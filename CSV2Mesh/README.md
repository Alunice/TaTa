# CSV to Mesh
![image](Raw/csv2mesh.png)

## 介绍:
* 这是一个将Renderdoc截帧出来的模型csv文件批量转换为UnityMesh的工具，且支持生成FBX格式文件。

![image](Raw/Show Mesh.gif)
